Common questions

Can I use my own project photos?
Yes. Your own photos make the site far stronger. Send us whatever you have and we use them. If you don't have professional photos yet, we use high-quality placeholder images while you build your library.
How is this different from Squarespace or Wix?
With Squarespace or Wix, you pick a template and build it yourself — that takes hours of your time and design skill. With us, you fill in one short form and we build the entire website for you. You see the finished result before deciding anything.
Do I need a domain name already?
No. You see your website on a preview link first. If you want to go live, we help you set up a domain — or connect it to one you already have.
Will I be able to update the website myself?
Yes. Once you approve the site, we set it up so you can make basic updates yourself — add new project photos, update your services, change contact details. No coding needed.
